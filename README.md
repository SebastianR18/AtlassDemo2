# Atlass Demo P2

## Descripción
**Atlass Demo** es un proyecto desarrollado en Unity que presenta una experiencia de exploración y aventura en 2D, incorporando animaciones, personajes y mecánicas interactivas.

El proyecto está diseñado como una demostración jugable y puede ejecutarse directamente desde Unity o mediante una build exportada.

---

# Características

- Escenario 2D interactivo.
- Personajes animados.
- Sistema de animaciones y controladores.
- Integración de assets y sprites personalizados.
- Proyecto organizado para desarrollo en Unity.

---

# Requisitos

Antes de ejecutar el proyecto, asegúrate de tener instalado:

- Unity Hub
- Unity Editor (versión compatible con el proyecto)
- Git (opcional, para clonar el repositorio)

---

# Instalación

## Opción 1 — Descargar desde Itch.io

1. Ingresa a la página oficial del proyecto:

```text
https://sebastian819.itch.io/atlassdemo-p2
```

2. Descarga la versión más reciente.
3. Extrae el archivo `.zip`.
4. Ejecuta el archivo del juego correspondiente a tu sistema operativo.

---

## Opción 2 — Abrir el proyecto en Unity

1. Clona este repositorio o descarga el proyecto:

```bash
git clone https://github.com/SebastianR18/AtlassDemo2.git
```

2. Abre **Unity Hub**.
3. Selecciona **Add Project**.
4. Busca la carpeta del proyecto `AtlassDemo`.
5. Abre el proyecto con la versión recomendada de Unity.

---

# Ejecución

Una vez abierto el proyecto:

1. Abre la escena principal.
2. Presiona el botón **Play** dentro de Unity.

---
